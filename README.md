# Day 4 Assignment – Object-Oriented Programming (OOP)

## Overview

This repository contains my Day 4 Python assignment on Object-Oriented Programming (OOP). The objective of this assignment is to understand and implement the core principles of OOP using Python.

## Learning Objectives

* Understand Classes and Objects
* Use Constructors (`__init__`)
* Work with Instance and Class Variables
* Implement Encapsulation
* Apply Inheritance
* Demonstrate Polymorphism
* Understand Abstraction
* Use Method Overriding
* Implement `super()` Function
* Learn Access Modifiers (Public, Protected, Private)

## Topics Covered

### Theory Topics

5. Encapsulation
6. Inheritance
